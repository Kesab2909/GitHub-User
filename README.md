# 🔍 GitHub User Finder

A sleek, responsive React application to search GitHub users and view their public profile information. Built with modern React hooks and styled for simplicity, this app helps you search users, view details, and revisit recent searches.

---

## 🚀 Features

- 🔎 **Search by GitHub username**
- 👤 **Display user details**: Name, bio, avatar, followers, repos, etc.
- 🧠 **Recent search history** (stored in `localStorage`)
- ⚡ **Loading indicator** while fetching data
- 🚨 **Error handling** for invalid usernames
